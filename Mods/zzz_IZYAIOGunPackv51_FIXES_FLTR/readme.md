# Flatlander's Fixes for IZY-All in One Gun Pack v5.1 - version 2.5.1.0

## Overview
The mod provides various fixes for the IZY-All in One Gun Pack v5.1.

## List of fixes
 1. Level 6 RPD Machine Gun (T2) and Antique Machine Gun (T4) have the correct number of mod slots. 
 2. Box of HP 7.62mm Ammo (ammoBundle762mmBulletHP) uses the correct item icon.
 3. Recipes for ammo762mmExplosiveFragmentations and ammo762mmExplosiveFragmentationsBundle unlock at the same time as the recipe for IZYgunT4MarksManRifleM110SASS.
 4. Various localization fixes.

## Requirements
  - [IZY-All in One Gun Pack v5.1](https://www.nexusmods.com/7daystodie/mods/5458),
  - [z Izayo Weapon Fixes](https://www.nexusmods.com/7daystodie/mods/5458),
  - 7 Days to Die v2.5 and later.

## Manual installation
 1. Download the mod file.
 2. Unzip it anywhere on your hard drive.
 3. Copy the **zzz_IZYAIOGunPackv51_FIXES_FLTR** folder to the 7 Days to Die Mods folder. The default location is **%APPDATA%\7DaysToDie\Mods**.

## Important notes
 1. Adding the mod to an existing save is probably safe, but you should always make a backup of your game save.
 2. Removing the mod is probably safe, but you should always make a backup of your game save.
