# Give Me Slugs For This Pipe Shotgun! - version 2.6.1.0

## Overview
The **Give Me Slugs For This Pipe Shotgun!** mod replaces the 15 shotgun shells you receive in the Pipe Shotgun Bundle that the trader gives you for completing the Intro Buried Supplies quest with 10 shotgun slugs. Who can say no to more stopping power for the humble Pipe Shotgun?

## Requirements
 - 7 Days to Die v2.5 and later.

## Manual installation
 1. Download the mod file.
 2. Extract it anywhere on your hard drive.
 3. Copy the **GiveMeSlugsFLTR** folder to the 7 Days to Die Mods folder. The default location is **%APPDATA%\7DaysToDie\Mods**.
 4. Verify the install by checking that the ModInfo.xml file is located in this hierarchy: "\Mods\GiveMeSlugsFLTR\ModInfo.xml"

## Important notes
 1. Removing the mod is probably safe, but you should always make a backup of your game save.
 2. The mod is compatible with [Extreme Quick Start](https://www.nexusmods.com/7daystodie/mods/8856) and all my other mods which modify starting quests and starting quest rewards.
