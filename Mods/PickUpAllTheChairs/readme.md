# Pick Up All The Chairs - version 2.5.1.0

# Overview
**Pick Up All The Chairs** lets you pick up, scrap and place broken and overturned chairs (woodChair1Broken and oldChair1Broken) as well as plastic chairs and plastic chair stacks (schoolSeat01 and schoolSeat02).

# Requirements
 - 7 Days to Die v2.5 and later.

# Manual installation
 1. Download the mod file.
 2. Decompress it anywhere on your hard drive.
 3. Copy the **PickUpAllTheChairs** folder to the 7 Days to Die Mods folder. The default location is **%APPDATA%\7DaysToDie\Mods**.

# Important notes
 1. Removing the mod is probably safe, but you should always make a backup of your game save.
